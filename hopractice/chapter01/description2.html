Transmission Control Protocol id a method (protosol) used along with the Internet Protocol (IP) to send data in the form of a message units, called packets, between computers over the Internet.
IP
Internet protocol is the method or protocol by which data is sent from one computer to another over the internet uniquely identified by an IP address.
FTP
File Transfer Protocol is a protocol used to exchange files between computers on the internet
